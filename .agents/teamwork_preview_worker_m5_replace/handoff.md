## 2. Logic Chain
- **Resolving PTA Missing Hits Test**:
  Since Garen has Stridebreaker, Mortal Reminder, and Phantom Dancer by default, his attack speed is high enough to execute 3 auto-attacks in under 1.5 seconds. The test `test_pta_missing_hits_garen` aims to verify that PTA does not trigger with only 2 attacks. Changing the test APL constraint in `tests/tier1_feature.rs` to `time<1.0` limits Garen to exactly 2 basic attacks, making the test pass successfully and genuinely verifying the 3-attack requirement.
- **Fixing Clippy Warnings in lol-core**:
  - We collapsed the nested `if` statements in `crates/lol-core/src/buff.rs` using standard Rust `is_some_and`.
  - We collapsed the nested `if` statement in `crates/lol-core/src/champion.rs` by combining the outer and inner conditions with `&&`.
  - We refactored `crates/lol-core/src/event.rs` to tuple-match the event and the recorder option `(event, &ctx.recorder)` together, collapsing the nested `if let` blocks cleanly without needing nightly features like `let_chains`.
  - We replaced `self.current.armor = self.current.armor * ...` with `self.current.armor *= ...` in `crates/lol-core/src/stats.rs`.
  - We flattened the nested `mod tests` inside `crates/lol-core/src/item.rs` into the outer `mod tests` module block.
  These steps successfully reduced clippy warnings for `lol-core` and `lol-champions` to exactly 0.
